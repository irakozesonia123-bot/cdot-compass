# Brand assets

## CDOT / Colorado logo

To show the official logo in the top-left header, drop a PNG here named exactly:

```
cdot-logo-placeholder.png
```

No code changes are needed — `BrandLogo` resolves the file automatically
(`src/components/common/BrandLogo.tsx`). Until a file is present, a polished
on-brand fallback (compass icon + "CDOT") is shown instead.

Recommended: a square, transparent-background PNG (e.g. 256×256).
